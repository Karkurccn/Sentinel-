# Sentinel Logistic Truck Enterprise Architecture

Repositorio oficial de arquitectura para **Sentinel Logistic Truck (SLEA v1.0)**.

Este repositorio documenta la arquitectura completa del producto:

- Fundamentos
- Reglas globales
- Módulos Core
- APIs
- Eventos
- Modelos de datos
- Prompts para Cursor/Kiro
- Roadmap

## Filosofía

Sentinel Logistic Truck es una plataforma logística empresarial construida bajo:

- Modular Monolith
- API First
- Domain Driven Design
- Metadata Driven
- Event Driven interno
- Multi-Tenant
- Multi-Company
- Storage desacoplado
